Prep for Converting SCSS to CSS using SASS
1.install ruby for windows
2.install dart sass in root directory C: from https://github.com/sass/dart-sass/releases/tag/1.49.7
3.add dart sass to path https://katiek2.github.io/path-doc/ 
4.create folder and file for css based on corresponding scss files
5.go to folder where scss and css folder are and then enter only "cmd" in the breadcrumb/file directory bar to open terminal.

Converting SCSS to CSS using SASS.
6.put in "sass --watch scss/style.scss:css/style.css" see https://sass-lang.com/guide for reference
